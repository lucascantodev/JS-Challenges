Money Manager

Write a JavaScript program that starts by asking for the initial amount of money available.

Options:
  1. Add money
  2. Remove money
  3. Exit
If the user chooses "Add money":
  • Asks for the amount to be removed;
  • Validates whether the value is valid (positive and doesn't exceed the balance);
  • Subtracts that value from the initial amount if it's valid;
  • Displays the menu again with the updated amount of money.
If the user chooses "Remove money":
  • Asks for the amount to be added;
  • Adds that value to the initial amount;
  • Displays the menu again with the updated amount of money.
If the user chooses "Exit":
  • Terminates the program.

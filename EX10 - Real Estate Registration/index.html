Real Estate Registration

Write a JavaScript program that serves as a real estate registration system and meets the following requirements:

  1. It should have an interactive menu that is always displayed until the user chooses to exit;
  2. The interactive menu should display the number of registered properties at the top;
  3. The menu should have an option to save a property;
  4. To save a new property, the program should request the following information:
    • Owner's name
    • Number of bedrooms
    • Number of bathrooms
    • Whether it has a garage
  5. The menu should also have an option to display all saved properties.

The program should only be terminated when the "Exit" option is chosen; otherwise, it should return to the menu.

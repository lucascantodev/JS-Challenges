Job Vacancy System

Write a JavaScript program that simulates a job vacancy system, where you can manage job listings and add candidates to those listings. It should meet the following requirements:

Have a menu where you can choose from different functionalities of the system:

  1. List available job vacancies
    • The option to list job vacancies should display the index, name, and the number of registered candidates for all job vacancies.

  2. Create a new job vacancy
    • The option to create a new job vacancy should ask for a name, description, and a deadline for the vacancy. It should also ask the user to confirm the information before saving it.

  3. View a job vacancy
    • The option to view a job vacancy should ask for the index of the vacancy and display all its information: index, name, description, deadline, number of candidates, and the names of the candidates.

  4. Register a candidate for a vacancy
    • The option to register a candidate for a vacancy should ask for the candidate's name, the vacancy index, and then display a confirmation with the vacancy's information before saving the candidate for the vacancy.

  5. Delete a vacancy
    • The option to delete a vacancy should ask for the index of the vacancy, display its information, and ask the user to confirm the deletion before actually deleting it.

  6. Exit

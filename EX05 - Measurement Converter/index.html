Measurement Converter

Write a JavaScript program that functions as a measurement converter. The program should ask for a value in meters and then give the option to choose to which unit of measurement this value should be converted.:

The options are:
  • Millimeter (mm);
  • Centimeter (cm);
  • Decimeter (dm);
  • Decameter (dam);
  • Hectometer (hm);
  • Kilometer (km);

The program should then convert the measurement according to the chosen option and display the result.

The program should also display an "Invalid option" message if the user enters an option different from the ones available (use break and default for this).

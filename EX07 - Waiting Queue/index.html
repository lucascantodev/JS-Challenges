Waiting Queue

Write a JavaScript program to simulate a waiting queue in a medical office.

The program should start by displaying an interactive menu on the screen with the following options:

  1. List of patients waiting in order, showing their position next to the name (e.g., 1st Matheus, 2nd Marcos, etc).
  2. Add a new patient to the end of the queue (asking for the patient's name).
  3. Consult the next patient in the queue and display the name of the consulted patient.
  4. Exit the program

The program should only be terminated when the "Exit" option is chosen; otherwise, it should return to the menu.

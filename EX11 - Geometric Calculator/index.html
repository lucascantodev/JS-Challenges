Geometric Calculator

Write a JavaScript program to calculate the area of different geometric shapes. The program should start with a menu containing various calculation options. The options should include:

  1. Triangle area: base * height / 2;
  2. Rectangle area: base * height;
  3. Square area: side²;
  4. Trapezoid area: (larger base + smaller base) * height / 2;
  5. Circle area: pi * radius² (consider pi = 3.14).

You should write the program using functions whenever possible to separate procedures. The program should also have an "Exit" option, and until it is chosen, it should return to the menu.
